## Hi, I'm Mariana 👋

Software developer with experience in Laravel, .NET and Vue, 
and a growing interest in Data Engineering and data pipelines. 
I enjoy moving between building applications and processing 
the data they generate.

### 🚀 Featured Projects

- **[auto-artigo](https://github.com/marianaflima/autoartigo-autism)** — Automated ETL pipeline built with Google Apps Script 
  that collects metadata from health-related academic articles via API, 
  cleans and structures the data into a spreadsheet. Runs monthly through 
  configured triggers, with no manual intervention.

- **[dpt-movie-lens](https://github.com/marianaflima/dpt-movie-lens)** — Data cleaning and modeling of the MovieLens 
  dataset using DuckDB and MinIO as the storage layer, orchestrated with 
  Docker, with views built for dashboard consumption in Metabase.
